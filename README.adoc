== Usage

Just add the maven coordinates to your dependency management configuration. For maven it'd be:

```xml
<dependency>
	<groupId>nz.co.aetheric</groupId>
	<artifactId>kottlefish</artifactId>
	<version>${version.kottlefish}</version>
</dependency>
```
